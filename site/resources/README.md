# Resources — drop images here

Put any images you want to use on the site in this folder. They're served at:

    https://classroomfaradaybags.com/resources/<filename>

Reference them in a page like:

    <img src="/resources/my-photo.jpg" alt="...">

Notes
- Supported: .jpg .png .webp .svg .gif
- Big photos: resize to ~1600px wide before adding (keeps pages fast).
- Everything here gets a 1-year browser cache automatically.
- Tell Claude the filename + which page to use it on, and it'll wire it in.
